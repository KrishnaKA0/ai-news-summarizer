# AI-Powered News Headline Summarizer 📰

An end-to-end Python pipeline that dynamically scrapes live headlines and processes them using a local AI NLP Transformer model.

## 🛠️ Tech Stack
- **Frontend UI:** Streamlit Cloud
- **Web Scraping:** BeautifulSoup4 & Requests
- **AI/NLP Model:** Hugging Face (T5-Small Transformer)
